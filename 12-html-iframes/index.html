HTML Iframes

An iframe is used to display a web page within a web page. The <iframe> tag specifies an inline frame.

Iframe - Remove the Border

By default, an iframe has a border around it. Use the border CSS property to change or remove it.

Iframe - Target for a Link

An iframe can be used as the target frame for a link. The target attribute of the link must refer to the name attribute of the iframe.
