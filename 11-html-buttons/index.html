HTML Buttons

Buttons let users interact with the web pages. The <button> tag defines a clickable button. By itself, the button does nothing until you add an action to it.

Disabled buttons

These buttons are not clickable.

Buttons with JavaScript

We can add behavior to buttons using the onclick attribute.

Button types

There are 3 types of buttons:

  • Button (default) - clickable
  • Submit - submits data to a form
  • Reset - resets the data in the form
